# Fitness for purpose and task effectiveness

`https://taxonomy.eticas.ai/risk/fitness-for-purpose`

**Maturity:** <span class="badge badge-emerging">emerging</span>

The system has not been validated against the problem it is intended to solve, or does not achieve adequate or equitable task performance across subpopulations and input types. Covers both the ex ante question (was this designed for this problem?) and the ex post question (does it actually work?).

> **This subcategory is emerging.** It has not yet been validated through established assessment methods.

**Also known as:** Fitness for purpose · Task success

**Applies to:** ALL  
**Lifecycle stages:** Pre Processing, In Processing
